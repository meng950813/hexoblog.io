鄙视题

jQuery

$.each(source,function(index,value){…}) VS $(…).each(function(index,element){…})

$.each 是 jQuery 全局方法,存储在jQuery全局对象上,主要遍历普通数组

$(…).each 是 jQuery 对象方法,存储在原型对象 jQuery.fn 上,主要用于对查询结果中的每个 DOM 元素执行操作
